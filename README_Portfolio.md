Executive Summary

An interactive dashboard designed to monitor and analyze customer sentiment towards major US Airlines in real-time. This project solves the problem of unstructured social media data by transforming 14,000+ raw tweets into actionable operational insights using Python.

Business Value

Operational Efficiency: Identified peak complaint hours (Mondays, 2 PM - 5 PM) using Heatmaps to optimize Customer Service staffing.

Root Cause Analysis: Detected that "Late Flight" and "Cancelled" are 3x more frequent than "Price" issues, shifting the focus from pricing strategy to operational reliability.

Competitor Benchmarking: Benchmarked United vs. Virgin America to reveal disparate customer satisfaction levels.

Tech Stack

Data Processing: Python, Pandas, NumPy

Visualization: Plotly Express (Interactive Charts), Matplotlib, WordCloud

Web Framework: Streamlit

Statistics: Boxplot Analysis for Confidence Level Validation

Key Features

Interactive Filters: Multi-select airlines and dynamic date ranges.

Temporal Analysis: Heatmap visualization of complaint intensity by Day & Hour.

Text Mining: N-gram analysis/WordCloud to extract dominant complaint topics.
